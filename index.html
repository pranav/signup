Self Serve UNIX Groups.

Description

A major problem with CCIS infrastructure is that you can't manage your own UNIX groups. For example, for every person that we add to Crew, we had to ask Systems to add them to the offical UNIX group. Professors use UNIX groups for giving Tutors and TAs permission to homework assignments for grading. Each person has to go through Systems and it would be much better if there was a web interface so people could help themselves.

Things you will learn.

  • Linux Administration
  • Directory Services such as NIS and LDAP
  • Web Development in Ruby/Python/PHP/Node or whatever
  • How to interface with existing infrastructure
